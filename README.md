# GamePortal API

REST API serwisu z grami wzorowanego na gram.pl / gry-online.pl.
Projekt portfolio — Junior .NET Developer.

## Tech Stack

- .NET 10, ASP.NET Core Web API
- Entity Framework Core 10 + SQL Server
- ASP.NET Core Identity + JWT Bearer
- FluentValidation
- xUnit + Moq
- Docker + docker-compose
- Swagger / OpenAPI
